  bower install zyll/yacp

yacp deps :

  jquery, backbone and colorpicker (yep we use one, yacp purpose isnt to
  invent the wheel)

## Developers

### Dependencies

* node
* grunt

## Install

Run `npm install && npm test`

### Testing

Run `grunt` to execute the specs.

You can watch for file modfication with `grunt watch`.
